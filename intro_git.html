GIT

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

  • sistema de controlo de versões
  • gratuito;
  • open source

comando para ver se temos instalado e qual a versão: git --version

para iniciar um repositorio local: git init

para consultar o status do ficheiros: git status

para adicionar ficheiro ao staging area/area de preparaçao: git add intro_git.html

para inicializar:git commit -m "commit inicial"

para adicionar todos os ficheiros: git add .

quand for para guardar uma versão do programa / fazer commit /efetuar snapshot: git commit -m "pequna descriçao"


para ver o historico de commits: git log

para ver o historico de commits com grafico: git log --graph

para ver o historico de commits numa só linha: git log --oneline

Ramificação do código

criar um ramo novo: git branch nome

mudar de ramo : git switch 'nome'

fazer o add e o commit com as novas alteraçoes no novo ramo

merge como o do outro branch : git merge nome

eliminar ramo:git branch -d nome

ver diferencas entre versoesgit diff
